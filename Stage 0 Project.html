Tags

There are several tags that are used in HTML.

  1. b for BOLD
  2. em for Italics
  3. img for an image


You can bold something right in the middle of the sentence and also italicize the whole sentence.
If you happen to forget a closing tag then everything after the open tag will be that continuously.
Images don't have closing tags, these tags are called void.

HTML Attributes


Attributes always equal a value.
An attribute is a property of an HTML element.
Example:
href and src are attributes.
href create links and src puts an image on the page. If you put text in front and behind the img tag it will just place the image inside of it.

Whitespace


If you place a br inside the tag <> symbols you can break a line up from being one long string of words. The computer doesn't know that you need a new line so you must put in the br inside <>.
The br tag is also a void tag as you don't need to put a closing tag.
All you have to do is enter br in <>
anywhere you want to start a new line and it will start a new line for you just like this sentence.
